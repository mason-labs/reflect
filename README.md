# reflect-site

Pulp sci-fi marketing site for the [reflect agent-skills package](https://github.com/ckorhonen/reflect).

Live at [cdd.dev/reflect](https://cdd.dev/reflect/) and [madewithmason.com/mason-os/reflect](https://madewithmason.com/mason-os/reflect/).

Cloudflare Worker + static assets, no build step. Deploy with `wrangler deploy`.
Cover art generated with gpt-image-1.5 (prompts in the repo history).
